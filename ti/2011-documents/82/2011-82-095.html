    reCaptcha Action Plugin 1.2.0.0

    reCaptcha Action Plugin 1.2.0.0
    1.2.0.0 is officially LunaStudios supported


    After a meeting me and infinityscape have worked out all the seeable bugs
    and now offer you this new version of the plugin.

    Function List.
    Recaptcha.SetController
    (Enables you to set your reCaptcha Keys)
    Recaptcha.Create
    (Creates the image)
    Recaptcha.Authorize
    (Checks the users input with the reCaptcha)
    Recaptcha.Cleanup
    (Cleans up the files off the users computer)


    This plugin could be used to stop micro's/Bots but it will also could be used to make sure someone is fully connected to the internet and also it could be used for register frount end pages for websites that have been inbeded in to ams.

    This plugin was by the Luna Studios Dev Team and any help then please
    contact use via here or the email address provided in the plugin.
