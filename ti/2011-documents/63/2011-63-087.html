  15. Michael Anderson said:

    Michael Anderson

    The author's mobile phone-based application of crowdsourcing implements Brenham's call for justice by providing a source of income for workers in developing countries where unemployment approaches 50%. Noting that in East Africa, 90% of the population has access to mobile reception and that the vast majority of mobile phone subscribers, "live in the developing world," Eagle first created a mechanism to gather blood bank levels in Kenyan hospitals via text message in order to more effectively manage blood supplies.

    The experiment initially failed until motivation was provided to nurses in the form of airtime credits, a payment which serves as, "a surrogate for currency." A subsequent implementation using USSD messaging which has no associated airtime charges is pending. While the cost to the crowd will be eliminated in this new iteration, some motivational factor will likely be required to achieve widespread participation.

    In order to deal with the issue of accuracy, Eagle extended the expectation maximization model to estimate and account for reporting bias. The author incorporated task repetition through a majority voting algorithm and found that quality approached the 98% level when the number of users exceeded 13. However, Eagle extended the majority voting approach by incorporating summative measures to infer individual accuracy based on task type which allowed his model to estimate any given user's accuracy and expertise. It is unclear if expertise is assigned greater weight in the data aggregation.

    Eagle surveys several tasks which seem appropriate for a crowdsourcing approach—transcription and localization, journalism, search refinement, sentiment and market research—and describes implementations within specific user scenarios. While the model is still for-profit, the overarching ethical goal provides an effective realization for Brenham's problem-solving approach in non-profit initiatives.
