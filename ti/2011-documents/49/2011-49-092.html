PHP - What Is Inn The ReCaptcha Script The 'is_valid' Text? Is It An Array Key?

Please don't mind my use of English. In the reCaptcha script you'll find this piece of code:

$recaptcha_response->is_valid = false;

I wonder what the -> does/means. Is it an alternative way to assign an key (with a value) to a string?

This question and answers originated from stackoverflow.com
Question by Ben

Other Answers

Answer >> Re: What Is Inn The ReCaptcha Script The 'is_valid' Text? Is It An Array Key?


That's actually php's object notation, you should see the object $recaptcha_response declared somewhere else in the script. It is calling the is_valid method of that object/class.

Answer >> Re: What Is Inn The ReCaptcha Script The 'is_valid' Text? Is It An Array Key?


$recaptcha_response is an instance of a object.

is_valid is a public property of the object.

$recaptcha_response-> = false assign a boolean values (false) at the property is_valid.

Later you can use it like if ($recaptcha_response->is_valid)//Do something
