It should be installed...I think you need to select Enable reCAPTCHA element - Yes
from Administration -> Users -> Authentication -> Email-based self-registration

and enter ReCAPTCHA public key or ReCAPTCHA private key in

Administration -> Users -> Authentication -> Manage authentication

Re: Does anyone know how to make reCaptcha work on Moodle 1.9?
by Helen Foster - Thursday, 3 April 2008, 05:43 PM
Hi Robert,

Yes, ReCAPTCHA is included in the latest Moodle 1.9+.

In addition to setting reCAPTCHA keys and enabling the reCAPTCHA element in the Email-based self-registration settings, as Mauno describes, email-based self-registration should be set as the self registration authentication plugin in the manage authentication common settings.

Re: Does anyone know how to make reCaptcha work on Moodle 1.9?
by Dennis Fulton - Wednesday, 29 September 2010, 08:40 PM
 
I found these steps to be helpful in clearing up why I wasn't able to get reCAPTCHA working.

  1. Go to http://www.google.com/recaptcha and register. Copy your public and private keys.
  2. In your Moodle Site go to Administration -> Authentication -> Email-based self-registration
  3. Select YES to Enable reCAPTCHA element and save changes
  4. Under the MANAGE authentication menu scroll down to the bottom of the page and enert the ReCAPTCHA public key and the ReCAPTCHA private key. Then save changes.
