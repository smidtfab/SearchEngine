Yes, I both use and find html easy to use.

When learning html, there are really two essential asspects of it, that is the markup language itself (html) and the presentation "language" (css, cascading style sheets).

html, use to have the presentation (colors, font style etc...) as part of the markup, but with css this has changed and for the better. Sticking to the topic, a simple html page can easily be made like thus,

<html>
<body>
<p>
<p>Hi, I'm a paragraph, and I'll look like a paragraph in the web browser.
I can <b>&lt;b&gt;bold&lt;/b&gt;</b> and <i>&lt;i&gt;italics&lt;/i&gt;</i>.
I can also <u>&lt;u&gt;underline&lt;/u&gt;</u>.
</p>
</p>
</body>
</html>

Technically, this isn't a "legal" html page, because you need a doctype at the top, but it will work! The easiest way is to create a template html file, which has the doctype you want to use, and then you won't have to redo this everytime.

A doctype simply tells the browser which version of html or xhtml you want to use.
