evly.com - First Crowdsourcing Social Network  

We want to provide people with the tools to easily build their own crowdsourcing websites and a platform to connect to a wide social network

Project Name
evly.com
Category
Website builder/Social Network
Details
evly is the first crowdsourcing social network that utilises a customised drag-and-drop website builder that allows registered users to build their own crowdsourcing websites for free. It also provides a social networking platform where users can interact with each other and reach interest-focused communities to source the best solutions from the right/specific target markets.

evly is currently in its beta phase where only a randomly selected user group is able to test the website builder. It is however open to the public for participation and contributing to current crowdsourcing sites and initiatives on the evly network.
How to Participate
Go to http://www.evly.com to sign up and start participating with different crowdsourcing websites and projects.

As evly is currently in it's beta phase, only a limited number of randomly selected users will receive a special build token that will allow them to start building their crowdsourcing websites. In order to be added to the waiting list to receive a token, email buildtoken@evly.com to request your token.
