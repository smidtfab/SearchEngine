Updates to the Mechanical Turk API

We've just rolled out some enhancements to the Mechanical Turk API; the new API version is 2006-01-16.

New features include the use of HIT Types to simplify the process of creating new HITs, better handling of reviewable HITs, the ability to update a Qualification test, and additional information provided to the Requester as part of a Qualification request from a potential worker.

More information can be found in the discussion board post.
