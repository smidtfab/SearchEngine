reCAPTCHA Plugin

Configuration Instructions
You must register with the reCAPTCHA project in order to use this plugin. They will generate a public/private key pair which you will use below.

  1. Open up the configuration screen for the reCAPTCHA plugin
  2. Copy and paste the public key into the "Public Key" text box
  3. Copy and paste the private key into the "Private Key" text box
  4. If you want users who are registering themselves to solve a reCAPTCHA as part of the registration process, enable the "Registration?" checkbox
  5. If you want users to solve a reCAPTCHA prior to submitting a comment on a blog post, enable the "Comment?" checkbox
  6. If you want users to solve a reCAPTCHA prior to sending a message through your blog to you or another user, enable the "Email?" checkbox
  7. If you want registered users who are logged in to avoid having to solve a reCAPTCHA, enable the "Exempt Users?" checkbox
  8. If your blog is operating over SSL and you want the reCAPTCHA to do the same, enable the "Enable SSL" checkbox; you may run into problems if you enable this option and your blog is not SSL enabled!
  9. If you want to change up the apperence of the reCAPTCHA, you can theme it using the options below, but you will have to enable the "Apply Theme?" checkbox to apply those options
  10. Theme: one of White, Red, Blackglass, or Clean
  11. Tab Index: enable users to your blog to use tabs to navigate elements on your page
  12. Language: reCAPTCHA can be displayed in one of eight languages: English, Dutch, French, German, Portuguese, Russian, Spanish, or Turkish
