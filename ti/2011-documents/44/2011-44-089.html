api-categories

Get Categories of a Session

Resource URI

/categories

Request (GET params)

Property Description
key key to access API
session session id received during create session
num_categories max categories to return (default 10)
nuggets true: if you want top nuggets of each category.
Do not pass this parameter if you want only categories.
num_nuggets_per_category used only if nuggets=true (default 5)
search return categories with given topics (default all categories)
url return categories of only given document (default all categories)
category_id return categories related to given category (default all categories)
pretty will return pretty JSON

Response (JSON)

Property Description
categories JSON object with category data (see example)

Examples

Get only categories

GET /categories?key=test&session=1234567890&pretty=1
